# Skill: Agent Behavior Standards

## Context
Use this skill to define how AI agents and human contributors should interact with the codebase, document changes, and follow coding conventions. It ensures consistency across all development cycles.

## Guidelines
- Follow standardized JSDoc for all methods.
- Maintain version consistency across all project files.
- Adhere to the multi-tenant isolation testing protocol.
- Use the standard ERP module structure for all new features.
